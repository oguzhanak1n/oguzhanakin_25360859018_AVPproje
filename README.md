# oguzhanakin_25360859018_AVPproje
Algoritmalar Ve Programlama dersi projesi 

Bu C dilinde yazılmış konsol tabanlı bir uygulamadır ve çeşitli fizik deneylerinin farklı gezgenlerdeki verilerini görüntülemeyi sağlar.Tamamen benim tarafımdan yapılmıştır.

Açılış ekranında isim girilmeli, menüden istenilen deney seçilmeli, programın sizden istediği veriler belirtilen türlerde(kg,m,vb.) girilmelidir ve sonuçlar ekrana yansıtılmaktadır. Menü ekranından çıkmak için "-1" yazılabilir bu programı kapatacaktır.
